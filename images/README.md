## How to Upload Photos

### Main Profile Photo
Upload your favorite photo of Bhawna as:
- `images/bhawna.jpg` or `images/bhawna.png`

This will appear in the circular photo frame on the main page.

### Gallery Photos
Upload multiple photos to the `gallery` subfolder:
- `images/gallery/photo1.jpg`
- `images/gallery/photo2.jpg`
- `images/gallery/photo3.jpg`
- etc.

These will appear in the photo gallery section.

## Supported Formats
- JPG/JPEG
- PNG
- WebP

## Tips
- Use high-quality images for best results
- Recommended size for main photo: 500x500 pixels or larger
- Gallery photos: Any size, but consistent dimensions look best
- Keep file sizes reasonable (under 2MB each) for faster loading
